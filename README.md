# State occupancy in covariate-driven HMMs

This repository contains the R code for the paper ["Inference on state occupancy in covariate-driven hidden Markov models"](https://arxiv.org/abs/2512.17496). 

In the paper, we develop two approaches for obtaining the state occupancy distribution of covariate-driven hidden Markov models (HMMs) as a function of a covariate of interest while accounting for the dynamics of the covariate process. We conduct simulation experiments and a case study on giant Galápagos tortoises, to demonstrate feasibility and applicability.
